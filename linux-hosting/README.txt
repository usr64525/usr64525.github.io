These are configuration files suitable for e.g. a VirtualBox server run from a Linux/Windows/macOS host.
The VM is setup with 2 VHDs, for example, one fifteen-gigabyte "main" and another hundred-gigabyte VHD for the webserver.
Debian, 64-bit, 1 or 2 CPUs, 512MB - 2GB RAM, 1 or 2 bridged NetworkInterfaceCards. Inside the VM, an unique static ipv4 IP is setup to be separate from the host IP.
